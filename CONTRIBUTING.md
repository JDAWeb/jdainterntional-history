# Contributing to JDA International Legacy Site

**Thank you** for your interest in helping preserve 32 years of humanitarian work.

This website[](https://jdainternational.org) is the official public archive of Joint Development Associates International (1993–2025).  
It is now in **maintenance-only / archival mode**.

### What kind of contributions are welcome?

| Type                         | Welcome? | How to submit                                                                 |
|------------------------------|----------|-------------------------------------------------------------------------------|
| Typo fixes, broken link fixes| Yes      | Open a GitHub issue or submit a pull request                                  |
| Better photo captions        | Yes      | Open an issue with the suggested caption and photo name                       |
| Improved alt text            | Yes      | Same as above                                                                 |
| New historical photos / documents (with permission) | Yes | Contact Bob directly first (see below)                                        |
| Major redesigns or new pages | No       | The site is intentionally frozen to preserve the historical record as-is     |

### How to report an issue or suggest a change
1. Open an **Issue** on this repository  
   → https://github.com/JDAWeb/your-repo-name/issues/new  
2. Clearly describe the problem or suggestion  
3. Include links/screenshots if helpful

### Contact for significant contributions
For anything beyond small fixes (e.g. adding new historical material), please email first:

**administrator**  
Email: legacy@jdainternational.org

All contributions become part of the permanent public archive and will be reviewed with care.

Thank you for helping honor this 32-year legacy with accuracy and respect.
